 APT (Package Manager)

                

                    
Update daftar paket

                    

                        
sudo apt update

                    

                

                

                    
Upgrade semua paket

                    

                        
sudo apt upgrade

                    

                

                

                    
Install paket baru

                    

                        
sudo apt install <nama-paket>

                    

                

                

                    
Hapus paket

                    

                        
sudo apt remove <nama-paket>

                    

                

                

                    
Cari paket

                    

                        
apt search <nama-paket>

                    

                

            


            

                
 Systemd (Service Management)

                

                    
Cek status service

                    

                        
systemctl status <service>

                    

                

                

                    
Start service

                    

                        
sudo systemctl start <service>

                    

                

                

                    
Enable service saat boot

                    

                        
sudo systemctl enable <service>

                    

                

                

                    
Restart service

                    

                        
sudo systemctl restart <service>

                    

                

            


            

                
 Network

ip a

                    

                

                

                    
Lihat routing table

                    

                        
ip r

ping google.com

 File & Folder

                

                    
List file detail

                    

                        
ls -la

                    

                

                

                    
Copy file

                    

                        
cp source.txt destination.txt

                    

                

                

                    
Move/rename file

                    

                        
mv old.txt new.txt

                    

                

                

                    
Hapus file/folder recursive

                    

                        
rm -rf <folder>

                    

                

                

                    
Ubah permission

                    

                        
chmod 755 file.sh
